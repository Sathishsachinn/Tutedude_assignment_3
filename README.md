Problem 1: Factorial of a Number

#Description
This program calculates the factorial of a number entered by the user.
The factorial logic is implemented using a function and a loop.

### Features

* Takes input from the user
* Uses a function named `factorial`
* Handles negative number input
* Displays the factorial result

### How It Works

1. The user enters a number.
2. The number is passed to the `factorial()` function.
3. The function calculates the factorial using a loop.
4. The result is returned and printed.

### Sample Output

Enter a number: 5
The factorial of 5 is: 120
